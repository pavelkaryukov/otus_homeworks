2-мерная разряженная бесконечная матрица заполненная значениями по-умолчанию.

Matrix class

хранятся только занятые элементы. Присвоение ячейки в значение по-умолчанию освобождает ячейку.

TValue- тип значений в ячейках матрицы
TDefaultValue- значение ячейки по умолчанию
